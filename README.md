# Cape Fur Seal Entanglement Analysis (2021–2025)
## Project Overview
This capstone project analyzes entanglement incidents involving Cape fur seals along the Namibian coast between 2021 and 2025.
## Data
The dataset contains observational records collected during routine monitoring activities along the Namibian coast.
## Methods
Data were cleaned and analyzed using R and tidyverse. Exploratory Data Analysis (EDA) and visualizations were used to identify patterns in entanglement incidents.
## Key Findings 
This project analyzes Cape fur seal entanglement incidents along the Namibian coast from 2021–2025. Using exploratory data analysis and visualization, the study identifies the most common debris causing entanglements and the seal age classes most affected. Results show fishing lines and packaging materials dominate incidents, with pups experiencing the highest number of entanglements.
